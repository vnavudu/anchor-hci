# Anchor — High-Fidelity Prototype

Anchor is a calm, minimalist women’s mental-health companion designed for quick relief and gentle reflection. This high-fidelity Human-Computer Interaction prototype focuses on discreet, supportive interactions that help users recenter during stressful moments.

## Screens
- **Home** – Welcomes the user and offers direct access to a calming session or journaling flow.
- **Calm Session** – Guides users to choose a short, medium, or long grounding experience tailored to the time they have.
- **Log** – Provides a lightweight journaling card where users can jot feelings and store reflections.

## Tech Stack
- Built with React (Vite) for responsive, component-driven interactions.
- Styled with Tailwind CSS to deliver a soft, blue-toned interface that reinforces calmness.
- Crafted as a high-fidelity HCI prototype to visualize the Anchor experience.

## Try It Yourself
No need to be a tech expert—follow these steps slowly and you’ll be fine:

1. Open the GitHub page for this project and press the green **Code** button, then choose **HTTPS** and copy the link that appears.
2. On your computer, open the **Terminal** (Mac) or **Command Prompt** (Windows). Paste this command, replacing the example link with the one you copied:
   ```
   git clone https://github.com/your-username/anchor-hci.git
   ```
   Press **Enter** and wait while the project downloads. When it finishes, move into the project by typing:
   ```
   cd anchor-hci
   ```
3. Install the tools the project needs by typing:
   ```
   npm install
   ```
   This may take a minute—just let it finish.
4. Start the prototype with:
   ```
   npm run dev
   ```
   You’ll see a web link appear in the terminal. Hold the **Command** key (Mac) or **Ctrl** key (Windows) and click the link to open the Anchor prototype in your browser.

If anything looks confusing, go step by step and double-check your typing. You’ve got this!
